ScrumDesk
ScrumDesk is professional scrum project management tool helping agile teams to self-organize and product owners to plan products. The API lets users access data, edit them, create customized reports, for integration & synchronization with 3rd party tools. API is hosted, but it can be installed in the users&#039;s environment as well. It uses RESTful calls and supports XML, JSON and ATOM response formats.
