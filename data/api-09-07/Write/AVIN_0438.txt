Wine tracking number repository
AVIN
The AVIN is a unique 13 digit number, which acts like an ISBN for books, but is used for tracking wine. The AVIN API is a service that allows automatic access to AVIN&#039;s up-to-date wine information. Information exposed through the API includes producers, wines, country and wine types. The API uses RESTful calls and responses are formatted in XML and JSON.
