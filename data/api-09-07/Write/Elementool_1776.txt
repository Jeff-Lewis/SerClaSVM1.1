Elementool
Elementool is a provider of web based project management tools for help desk, bug and defect tracking and time tracking tools. The Elementool API lets users access the main features of Elementool&#039;s bug tracking service via a thrid party application. The API can be used to view, update and submit issues to a user&#039;s account without logging in to Elementool&#039;s Web site. It uses SOAP calls and responses are formatted in XML.
