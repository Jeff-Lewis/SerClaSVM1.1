TuneFind
TuneFind contains an index of music and songs appearing in popular television shows and movies. Find a song, or see where an artist&#039;s music has been featured. The TuneFind API is a RESTful API that allows developers to create applications that access data from TuneFind&#039;s database. All data is returned in JSON or XML format. Access to the API is requires permission or licensing.
