Twitter web based client, url shortener and image sharing
br.st
Br.st is a Twitter web-based client. It offers functionality such as automatically shortens links as you type them, shortens text, upload and add to a tweet images and files up to 10 MB each, supports Find People to Follow or add to a List etc. API code examples are available for VB.NET, C#, PHP and AS3 (Flex). An API key is required.
