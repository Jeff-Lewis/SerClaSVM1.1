Z33K Tournament API
Z33K is an online platform that facilitates organized competitive gaming. The site provides access to thousands of tournaments for gamers, live streaming video, forums, and coaching services. Z33K&#039;s free custom tools allow individuals and organizations to run all aspects of online tournaments including player registration and management, match reporting and live bracket generation. The API gives developers programmatic access to online tournament data. It uses HTTP calls and responses are formatted in XML and JSON.
