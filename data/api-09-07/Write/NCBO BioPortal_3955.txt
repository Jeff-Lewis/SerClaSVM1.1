NCBO Ontology Tools
NCBO BioPortal
The NCBO (National Center for Biomedical Ontology) was created to support biomedical researchers in their knowledge-intensive work, by providing online tools and a Web portal enabling them to access, review, and integrate disparate ontological resources in all aspects of biomedical investigation and clinical practice. One service the NCBO probides is a RESTful API for creating custom applications and user interfaces for retrieving information about ontologies stored by NCBO.
