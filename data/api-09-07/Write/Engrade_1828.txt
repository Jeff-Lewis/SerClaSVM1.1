Online Gradebook and Classroom Management service
Engrade
Engrade connects more than 3.1 million administrators, teachers, students, and parents through online classroom tools. Engrade&#039;s communication and online learning platform includes a gradebook, secure messaging, homework calendar, online quizzes, flashcards, and third party applications. Engrade&#039;s API allows developers to integrate class management tools with student information systems and partner services. The API uses RESTful calls and responses are formatted in XML, JSON and serialized PHP.
