Online web editor
jsFiddle
JsFiddle is an online web development editor, a playground for web developers. The tool can be used in many ways, such as an online editor for HTML, CSS, and JavaScript. Developers add their bits of code into boxes for each type. These snippets are called &quot;fiddles&quot; in the jsFiddle system. The jsFiddle API provides access to the fiddles created by the jsFiddle users.
