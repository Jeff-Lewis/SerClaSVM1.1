Minneapolis/St. Paul Metro Transit
Metro Transit of Minneapolis/St. Paul is the online portal to the Twin Cities&#039; network of buses, light rail, commuter trains, and other transportation resources. An API to access route, bus direction, and bus stop information, as well as bus schedule times. The API is available through Appspot. It is a RESTful API which returns responses in JSON format.
