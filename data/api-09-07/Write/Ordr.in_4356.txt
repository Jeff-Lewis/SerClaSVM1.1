Ordr.in
Ordr.in is a service to help connect restaurants to hungry customers. It helps them find places, determine menus and place delivery orders. The Ordr.in API is a RESTful API that allows developers to easily integrate dozens of restaurants into a single app. The Ordr.in API also lets app developers get paid every time their app generates an order.
