Domain Name WHOIS Service
Online Whois Lookup
Online Whois Lookup is a service that allows users to input domain name and see a domain&#039;s whois record. The Whois API can be used to check if a domain name is available for registration. It can also lookup information about when a domain was created, changed and its expiration date. The API uses HTTP calls and responses are formatted in XML and JSON.
