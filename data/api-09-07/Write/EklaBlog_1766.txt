EklaBlog
EklaBlog is a French blogging site that is both ad-free and free to use. In addition to text-based blogs, EklaBlog also supports photo-blogs and vlogs. The EklaBlog API provides an alternate way of updating one&#039;s blog that can be used from mobile devices.

The site and API documentation are all in French.
