miRNA database access service
miRMaid
miRMaid provides programmatic access to data in miRBase, a major database for microRNA (abbreviated miRNA) information. miRNA is a short ribonucleic acid (RNA) molecule found in eukaryotic cells. The miRBase database contains published miRNA sequences as well as annotations. miRMaid&#039;s REST API enables users to query miRBase and retrieve data in either HTML or XML format.
