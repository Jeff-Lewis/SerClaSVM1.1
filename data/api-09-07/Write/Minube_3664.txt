Minube
The Minube search API is an interface that lets programmers integrate minube.com searches and results into web sites, desktop applications, or mobile phone applications. Minube is a travel community that allows users to draw on others&#039; experiences to plan their own trip and share all of their travel experiences. More than 180 countries and 15,000 cities worldwide are represented. The API is focused on shared experiences, whether sites, reviews, photos and videos all the same, the API is designed to share any type of content generated by users.The API uses RESTful protocal and responses are formatted in either XML or JSON.
