Open Source Node.JS Platform as a Service
Nodester
Nodester provides Node.JS hosting, as platform-as-a-service. The platform is based on a RESTful API running in the cloud. The service supports Node.JS sandboxing of user apps,  which according the the site lets users &quot;install their own NPM modules using the Nodester APPNPM API.&quot; Nodester uses Git to push updates.
