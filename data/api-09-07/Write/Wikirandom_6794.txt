Generate Random Wikipedia Pages
Wikirandom
Wikirandom is a website were users can get 1-10 randomly chosen Wikipedia pages.  Wikirandom is build over the MediaWiki API, and provides it&#039;s own simple API. The WikiRandom API has only two methods, get random pages and get total count of all random pages generated through the API. The API is free to use and no authentication is required.
