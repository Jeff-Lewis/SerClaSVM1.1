Call center management service
Five9 
Five9 is a cloud computing platform for call centers. The Five9 Configuration Web Services API provides developers access to Five9 User, Calling List, Contact, and Campaign Management capabilities that can be leveraged by applications via SOAP. The Configuration Web Services API can be used to add new leads to lists, provision new Five9 users and manage campaigns within a 3rd-party application. Users must join the Developer Program to gain access to full API documentation including code samples.
