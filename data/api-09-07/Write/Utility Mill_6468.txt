Utility Mill
Utility Mill is a website that allows users to quickly create and update web utilities. Utilities are basic programs that can perform simple tasks, such as converting decimals to Roman numerals, or checking if an AIM user is online. The Utility Mill API allows developers to create applications around the Utility Mill utilities. Developers can access the directory of utilities, find the most popular, retrieve its description and even the Python code used to create the utility.
