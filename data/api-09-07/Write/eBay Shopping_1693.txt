The eBay Shopping API allows developers can programmatically interact with the eBay website. eBay Shopping Web Services lets developers query eBay for information, and receive data in XML, JSON, or eBay&#039;s Name Value format. 

The API allows users to re-created an eBay search without ever going on the eBay site, allowing integration with other applications and interfaces.
